Step-by-step guide to connect your application to the MySQL database

Configuration Guide: Follow these steps to connect your application

Database Connection Setup

To connect your application to the MySQL database, you need to configure your application with the database credentials. This process varies depending on your technology stack.

PHP
Node.js
Python

1

Create a Configuration File

Create a config.php file in your project to store database credentials:

// config.php
// Database configuration
$db_host = 'localhost';
$db_name = 'huile_de_chebe';
$db_user = 'your_username';
$db_pass = 'your_password';

// Create connection
try {
    $pdo = new PDO("mysql:host=$db_host;dbname=$db_name;charset=utf8", $db_user, $db_pass);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connected successfully";
} catch(PDOException $e) {
    echo "Connection failed: " . $e->getMessage();
}
2

Include the Configuration File

Include the configuration file in your PHP pages where you need database access:

// index.php or other PHP files
require_once 'config.php';

// Now you can use the $pdo object to query the database
$stmt = $pdo->query('SELECT * FROM products');
$products = $stmt->fetchAll(PDO::FETCH_ASSOC);

// Use the data in your application
foreach ($products as $product) {
    echo $product['name'] . ' - ' . $product['price'];
}
1

Install MySQL Package

Install the MySQL package for Node.js using npm:

npm install mysql2
2

Create a Database Connection File

Create a database.js file to handle database connections:

// database.js
const mysql = require('mysql2');

// Create connection pool
const pool = mysql.createPool({
    host: 'localhost',
    user: 'your_username',
    password: 'your_password',
    database: 'huile_de_chebe',
    waitForConnections: true,
    connectionLimit: 10,
    queueLimit: 0
});

// Get a promise based connection
const promisePool = pool.promise();

module.exports = promisePool;
3

Use the Database Connection

Use the database connection in your application:

// app.js or your route files
const db = require('./database');

async function getProducts() {
    try {
        const [rows, fields] = await db.execute('SELECT * FROM products');
        return rows;
    } catch (error) {
        console.error('Database error:', error);
        throw error;
    }
}

// Use the function
getProducts()
    .then(products => {
        console.log('Products:', products);
    })
    .catch(error => {
        console.error('Error:', error);
    });
Important Security Note

Never commit database credentials to version control. Use environment variables or configuration files that are excluded from your repository.

1

Create Environment Variables File

Create a .env file in your project root (add it to .gitignore):

DB_HOST=localhost
DB_NAME=huile_de_chebe
DB_USER=your_username
DB_PASS=your_password
DB_PORT=3306
2

Use Environment Variables in Your Code

Update your configuration to use environment variables:

// For PHP, use getenv() or $_ENV
$db_host = getenv('DB_HOST');
$db_name = getenv('DB_NAME');
$db_user = getenv('DB_USER');
$db_pass = getenv('DB_PASS');

// For Node.js, use process.env
const pool = mysql.createPool({
    host: process.env.DB_HOST,
    user: process.env.DB_USER,
    password: process.env.DB_PASS,
    database: process.env.DB_NAME
});
