Key Data Flow Connections

Cart Data: Items added in the Shop page are stored in localStorage and retrieved in the Shopping Cart page
Checkout Integration: Shopping Cart passes item data to Checkout page via localStorage
Payment Processing: Checkout page redirects to appropriate payment processor based on user selection
Receipt Generation: All payment pages generate receipts with consistent formatting and data
Cart Clearing: Successful payments automatically clear the cart from localStorage
